# Note_Reminder_App
A basic app of reminder with MVVm architecture that has following components-</br>
1-side navigation drawer from scratch</br>
2-4 fragments</br>
3-recycler view</br>
4-alarm manager</br>
5-swipe to delete note and drawable menus
